# Refuge Builder

[![Build](https://github.com/LarpCove/RefugeBuilder/workflows/Integration/badge.svg)](../../actions?query=workflow%3AIntegration) [![CodeCov Coverage](https://codecov.io/gh/LarpCove/RefugeBuilder/branch/main/graph/badge.svg)](https://codecov.io/gh/LarpCove/RefugeBuilder/branch/main)

Generates random builds for the Refuge larp to analyze for monster generation.

## Documentation & Blog

The documentation site is located here: https://larpcove.github.io/RefugeBuilder/

The raw files can be viewed from the [docs](docs) folder or built into a local site using Jekyll.
